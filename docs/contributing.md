# Contributing

Contributions are welcome.

## Ways to help

- Open an issue with a reproducible bug report
- Suggest improvements (especially around configuration ergonomics and generated code quality)
- Add examples under `docs/examples`
- Improve documentation (typos, clarity, missing topics)
- Submit a pull request

## When submitting a PR

- Keep changes small and focused
- Include a short explanation of the problem and the approach
- If you modify templates/generation logic, add/update relevant tests where possible
